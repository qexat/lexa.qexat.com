projects

Lexie

Lexie is a programming language inspired from OCaml and Rocq. Its type system is based on the Calculus of Constructions type theory, and as such, it is dependently-typed.

I have not formalized the grammar, so there is no example snippet yet.

Link: https://github.com/qexat/lexie
